Syarat & Ketentuan

Terakhir Diperbarui: 11 Desember 2025

1. Definisi Layanan

CRM Auto Intelligence ("Layanan") adalah perangkat lunak berbasis web (SaaS) yang menyediakan alat analisis data penjualan otomatis untuk membantu pemilik bisnis memahami kinerja toko online mereka. Dengan mendaftar atau menggunakan layanan kami, Anda menyetujui syarat-syarat ini sepenuhnya.

2. Akun & Keamanan

  • Autentikasi: Kami menggunakan layanan pihak ketiga (Clerk.com) untuk proses login yang aman. Anda bertanggung jawab penuh atas keamanan akun dan kerahasiaan kredensial Anda.
  • Kami tidak bertanggung jawab atas kerugian yang disebabkan oleh akses tidak sah akibat kelalaian Anda (seperti membagikan perangkat atau tidak logout di komputer umum).
  • Satu akun hanya boleh digunakan untuk satu entitas bisnis.

3. Pembayaran & Pengembalian Dana (Refund)

  • Biaya Langganan: Layanan ini berbayar. Status Premium diaktifkan secara otomatis berdasarkan pembayaran yang diverifikasi melalui sistem Scalev.
  • Perpanjangan Otomatis: Langganan Anda tidak diperpanjang secara otomatis (Manual Renewal). Jika masa aktif habis (sesuai data metadata akun Anda), fitur Premium akan terkunci otomatis.
  • Garansi Uang Kembali: Kami memberikan garansi uang kembali 7 hari untuk pengguna pertama kali jika layanan tidak berfungsi sebagaimana mestinya.

4. Batasan Penggunaan & Data

Anda setuju untuk tidak mengunggah data ilegal, berbahaya, atau melanggar hak cipta orang lain.

Format & Integritas Data:
Sistem kami menggunakan metode pemrosesan file CSV/JSON yang efisien (Bulk Processing). Kami tidak menyediakan fitur untuk menyunting data satu per satu di dalam aplikasi untuk menjaga integritas file asli Anda.

Jika terjadi kesalahan data atau Anda ingin memfilter pelanggan tertentu, Anda disarankan untuk menyunting file sumber (Excel/CSV) di komputer Anda, lalu mengunggah ulang file tersebut untuk memperbarui dashboard.

5. Hak Anda (Kontrol Penuh)

Anda memiliki hak untuk menghapus data Anda kapan saja.

Keamanan Penghapusan:
Saat Anda memilih fitur "Hapus Semua Data", sistem kami akan mengeksekusi perintah penghapusan di Server Storage dan Database sekaligus. File fisik akan dimusnahkan dan link akses akan dihapus, sehingga data tersebut tidak dapat dipulihkan kembali oleh siapapun.

Catatan: Penghapusan ini hanya memusnahkan data lama untuk menjaga privasi. Akun Anda tetap aktif dan Anda dapat mengunggah (upload) data baru kapan saja jika Anda ingin memulai analisis kembali.

6. Penafian (Disclaimer)

Layanan ini disediakan "sebagaimana adanya". Kami menjamin keamanan penyimpanan data sesuai standar enkripsi modern (RLS Supabase & Clerk Auth), namun keakuratan analisis bisnis bergantung sepenuhnya pada validitas data yang Anda unggah.

7. Perubahan Ketentuan

Kami dapat merevisi syarat dan ketentuan ini sewaktu-waktu. Perubahan signifikan akan diinformasikan melalui email atau notifikasi di dashboard.
