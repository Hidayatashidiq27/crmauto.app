Kebijakan Privasi

Terakhir Diperbarui: 10 Desember 2025

1. Pendahuluan

Selamat datang di CRM Auto Intelligence ("Kami", "Aplikasi"). Kami menghargai privasi data bisnis Anda. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data pribadi serta data penjualan yang Anda unggah ke platform kami (app.crmauto.id).

2. Data yang Kami Kumpulkan

  • Informasi Akun: Alamat Email, Nama, dan Foto Profil (yang didapatkan melalui Google Sign-In atau pendaftaran Email).
  • Data Bisnis (File CSV): File data penjualan (seperti dari Scalev, OrderOnline) dan data iklan (Ads Manager) yang Anda unggah secara sukarela untuk keperluan analisis.
  • Status Langganan: Data mengenai status pembayaran dan masa aktif akun Anda.

3. Penyimpanan & Keamanan Data (Teknis)

Kami menggunakan infrastruktur cloud standar industri untuk menjaga keamanan data Anda:

  • Google Firebase (Authentication & Database): Kami menggunakan Firebase untuk proses Login yang aman. Alamat Email Anda digunakan sebagai Unique Identifier (Kunci Utama) untuk memastikan bahwa hanya Anda yang dapat mengakses dashboard dan data langganan Anda.
  • Supabase (File Storage): File CSV yang Anda unggah disimpan di Supabase Storage yang terenkripsi.
    Kami menggunakan teknologi Signed URL (Tautan Terenkripsi Berwaktu) yang memastikan file Anda tidak dapat diakses oleh publik secara sembarangan tanpa otorisasi login yang sah.

4. Bagaimana Kami Menggunakan Data Anda

Data yang Anda berikan hanya digunakan untuk:

  • Menampilkan visualisasi grafik dan analisis profit di Dashboard Anda.
  • Memverifikasi status kepemilikan akun dan langganan (Pro/Free).
  • Memberikan rekomendasi strategi bisnis (jika Anda menggunakan fitur AI).

KAMI TIDAK AKAN PERNAH MENJUAL, MENYEBARKAN, ATAU MEMBAGIKAN DATA PELANGGAN/PENJUALAN ANDA KE PIHAK KETIGA MANAPUN.

5. Hak Anda (Kontrol Penuh)

Anda memiliki kendali penuh atas data Anda. Di dalam aplikasi, kami menyediakan fitur "Hapus Semua Data".

Jika Anda menekan tombol tersebut, sistem kami akan secara otomatis menghapus referensi file CSV Anda dari database kami, sehingga data tersebut tidak lagi dapat ditampilkan atau diakses.

6. Transaksi Pembayaran

Kami menggunakan layanan pihak ketiga (seperti Scalev/Payment Gateway) untuk memproses pembayaran langganan. Kami tidak menyimpan informasi Kartu Kredit atau data perbankan sensitif Anda di server kami.

7. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai kebijakan privasi ini atau ingin mengajukan penghapusan akun permanen, silakan hubungi kami melalui:

Email: support@crmauto.id
