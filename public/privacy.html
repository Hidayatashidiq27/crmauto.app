Kebijakan Privasi

Terakhir Diperbarui: 11 Desember 2025

1. Pendahuluan

Selamat datang di CRM Auto Intelligence ("Kami", "Aplikasi"). Kami menghargai privasi data bisnis Anda. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data pribadi serta data penjualan yang Anda unggah ke platform kami (app.crmauto.id).

2. Data yang Kami Kumpulkan

  • Informasi Akun: Alamat Email, Nama, dan Foto Profil (yang didapatkan melalui penyedia identitas Clerk.com).
  • Data Bisnis (File CSV/JSON): File data penjualan (seperti dari Scalev, OrderOnline) dan data iklan (Ads Manager) yang Anda unggah secara sukarela untuk keperluan analisis.
  • Status Langganan: Data metadata mengenai status pembayaran (Premium/Free) dan masa kedaluwarsa akun Anda.

3. Infrastruktur & Keamanan Data (Updated)

Kami telah meningkatkan standar keamanan kami dengan menggunakan infrastruktur modern yang terpisah untuk menjamin integritas data:

  • Autentikasi & Identitas (Clerk.com): Kami menggunakan layanan Clerk untuk manajemen akun yang aman. Data sensitif seperti status langganan (Premium) dan tanggal kedaluwarsa disimpan dalam Metadata Terenkripsi di server Clerk, bukan di database publik.
  • Database & Penyimpanan File (Supabase): Data penjualan dan file CSV Anda disimpan di server Supabase yang dilindungi dengan teknologi Row Level Security (RLS).

    Bagaimana Data Anda Dilindungi?
    • Akses Privat: Gudang penyimpanan file (Storage) kami dikonfigurasi secara privat. Akses publik (Anonymous) telah dinonaktifkan sepenuhnya.
    • Verifikasi Token: File Anda hanya dapat diunduh atau dilihat oleh sistem jika menyertakan "Token Digital" (JWT) yang valid milik Anda sendiri.
    • Isolasi Data: Sistem kami memvalidasi ID pengguna di level server sebelum mengizinkan akses apa pun, mencegah pengguna lain melihat data Anda.

4. Bagaimana Kami Menggunakan Data Anda

Data yang Anda berikan hanya digunakan untuk:

  • Menampilkan visualisasi grafik dan analisis profit di Dashboard Anda secara Real-time.
  • Memverifikasi status kepemilikan akun dan langganan (Pro/Free) melalui metadata Clerk.
  • Memberikan rekomendasi strategi bisnis.

KAMI TIDAK AKAN PERNAH MENJUAL, MENYEBARKAN, ATAU MEMBAGIKAN DATA PELANGGAN/PENJUALAN ANDA KE PIHAK KETIGA MANAPUN.

5. Hak Anda (Kontrol Penuh)

Anda memiliki kendali penuh atas data Anda. Di dalam aplikasi, kami menyediakan fitur "Hapus Semua Data".

Jika Anda menekan tombol tersebut, sistem kami akan secara otomatis menghapus file fisik dari Storage dan menghapus referensi link dari Database, sehingga data tersebut musnah secara permanen.

6. Transaksi Pembayaran

Kami menggunakan layanan pihak ketiga (Scalev) untuk memproses pembayaran langganan. Kami tidak menyimpan informasi Kartu Kredit atau data perbankan sensitif Anda di server kami. Status pembayaran Anda disinkronisasi secara otomatis ke akun Clerk Anda.

7. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai kebijakan privasi ini atau ingin mengajukan penghapusan akun permanen, silakan hubungi kami melalui:
